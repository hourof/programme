# 1.ST7735屏幕管脚意思

```
引脚功能对于说明
```

| 管脚 | 管脚其他称呼 | 管脚描述                     |
| ---- | ------------ | ---------------------------- |
| Gnd  |              | 接地                         |
| VCC  |              | 电源供电接口                 |
| SCL  | SCK          | SPI时钟线                    |
| SDA  | MOSI         | 主设备数据输出               |
| RES  | RST          | LCD的复位引脚                |
| DC   |              | 数据/命令切换                |
| CS   |              | 片选                         |
| BL   |              | 背光控制，gpio输出高电平点亮 |
